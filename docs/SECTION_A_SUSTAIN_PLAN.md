# Section A サステイン層の拡張メモ

- 目的: Section A の休止後にも電子層が呼吸し続け、Section B でビジュアル粒子が動き始めるための持続音を提供する。
- 方向性:
  - 初回のトーン cue 後、柔らかいサステインベッドを常時稼働させる。
  - 演奏者/シンセの各発音時に小さな「尾」を重ねて、休止中も細く音を繋ぐ。
  - 時間経過とともにサステインの質感を変化させ、Section B に入る頃にはノイズ成分が増える。
- 実装方針:
  1. サステイン専用の Faust DSP (`sustain_bed.dsp`) を用意し、サイン波からノイズまでテクスチャを連続的に変えられるようにする。
  2. `SectionAAudioSystem` にサステインベッド用のノード/ゲイン構成を追加し、ベースレベル・パルス加算・リバーブ送信を個別制御する。
  3. `CompositionPlayer` の Section A 制御からサステインベッドを起動・強化・質感推移させ、休止後〜再開〜B移行まで電子層が途切れないようにする。
  4. 演奏者 cue 度にベッドへ短いパルスを加えて「尾」を生み、進行ステージごとにノイズ寄りへフェードするロジックを組み込む。
- 期待効果:
  - 30秒休止後も背景に薄いサウンドが保たれ、ビジュアル側の粒子駆動が継続。
  - Section B へ向かうにつれて音色が徐々に変化し、最終的なホワイトノイズ的質感へ自然に到達する。
